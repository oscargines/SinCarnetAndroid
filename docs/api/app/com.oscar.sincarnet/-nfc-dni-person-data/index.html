NfcDniPersonData

internal data class NfcDniPersonData(val firstName: String, val lastName1: String, val lastName2: String, val documentNumber: String, val optionalData: String, val fatherName: String, val motherName: String, val birthDateAammdd: String, val birthPlace: String, val birthProvince: String, val residenceAddress: String, val residencePopulation: String, val residenceProvince: String, val nationality: String, val sex: String)

Datos de persona extraídos del DNI electrónico vía NFC.

Los datos se extraen de dos grupos de datos (DG) del MRZ (Machine Readable Zone):

  • DG1: Información básica (nombre, apellidos, fecha nacimiento, nacionalidad, sexo)

  • DG13: Información ampliada (padres, domicilio, provincia, lugar nacimiento)

Si DG13 no está disponible, algunos campos se rellenan con espacios vacíos.
