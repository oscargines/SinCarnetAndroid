printDiligenciaSuspend

private suspend fun printDiligenciaSuspend(context: <Error class: unknown class>, mac: String, jsonAssetPath: String, placeholders: Map<String, String>, investigado: PersonaInvestigadaData, sigs: PrintSignatures? = null, sharedConn: <Error class: unknown class>? = null)

Renderiza e imprime una diligencia de forma asincrónica.

Proceso:

  1. Lee el JSON de plantilla desde assets

  2. Resuelve todos los [placeholder] con datos reales

  3. Construye el body completo de la diligencia

  4. Envía a BluetoothPrinterUtils para conversión CPCL

  5. Imprime en la impresora Zebra

Manejo de QR: Si el documento incluye sección "qr", se usa printDocumentSuspend() que renderiza el código. Si no, se usa printDocumentResolvedSuspend().

Parameters

context

Contexto para Storage y Bluetooth

mac

Dirección MAC de la impresora

jsonAssetPath

Ruta en assets del JSON de plantilla (p. ej. "docs/02derechos.json")

placeholders

Mapa de [clave] → valor para reemplazar en la plantilla

investigado

Datos del investigado (para construcción de body)

sigs

Firmas capturadas (opcional, para atestados)

sharedConn

Conexión compartida Bluetooth (opcional, reutiliza de flujo anterior)
